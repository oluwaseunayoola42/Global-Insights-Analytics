# Global Insights Analytics

A data analytics project that uses **Power BI web scraping** to extract global population and demographic data, and visualizes it through an interactive Power BI dashboard.

## Dashboard Insights

- **Population by Year (1955–2026)** — global population has grown steadily from ~2.5bn to over 8bn.
- **Population by Country** — bubble map showing population distribution across all nations.
- **Population by Region** — choropleth map highlighting density across North America, Europe, Asia, Africa, and South America.
- **Population by Major Religious Groups:**
  - Christians — 2.17B
  - Muslims — 1.59B
  - Unaffiliated — 1.12B
  - Hindus — 1.03B
 



## Tools Used

- **Microsoft Power BI** — web scraping, data transformation, and dashboard visualization
